TSA (Test Searching Algorithms)
Version 1.1

test of the most popular searching algorithms
the collection includes the following algorithms:
	- linear search
	- binary search
	- jump search
	- interpolation search
	- exponential search
	- fibonacci search


to compile and run a standard assembly:
	make release
	./build/release/tsa


to compile and run an accelerated build:
	make release-thread
	./build/release-thread/tsa
